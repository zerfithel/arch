Complete installation guide:

> Install git
> - sudo pacman -S git

> Copy the repository
> - git clone []

> Go to your file localization
> - cd [localization/of/your/file]

> Build the package
> - makepkg -si

> Move it to $PATH
> - sudo mv /localization/of/your/pc /usr/local/bin/pc

> Make it executable
> - sudo chmod +x /usr/local/bin/pc

> List the commands:
> - sudo pc -h
